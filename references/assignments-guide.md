# 实操与作业指南

本项目不包含独立的作业仓库，使用课程中的 Hands-On 材料作为实操练习。

## 使用方式

- 看到材料标题包含 "Hands On" 时，视为实操任务
- 建议在 AWS Console 同步操作
- 完成后在 Obsidian 中记录：
  - 实操目标
  - 关键步骤
  - 遇到的问题与解决方式

## 交付建议

在 Vault 的 assignments/ 目录中按材料名创建记录，例如：

- 05 - EC2 Fundamentals - 003 Create an EC2 Instance with EC2 User Data to have a Website Hands On
- 08 - High Availability and Scalability ELB & ASG - 005 Application Load Balancer (ALB) - Hands On - Part 1

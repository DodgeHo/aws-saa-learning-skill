## 学习目标

- 通过动手操作创建 VPC、子网与基本路由，理解配置步骤与常见陷阱。

## 重点速览

- Hands On 涵盖：创建 VPC（选择 CIDR）、划分子网、配置路由表、创建 Internet Gateway 并关联、启动可分配公网 IP 的 EC2 实例。

## 详细内容

1. 步骤概览：创建 VPC -> 创建子网（跨 AZ）-> 创建/关联 IGW -> 更新路由表 -> 创建安全组 -> 启动 EC2 并分配公网 IP。
2. 注意事项：子网与路由表的关联，安全组默认规则，以及为生产环境禁用宽松的入站规则。

## 自测问题

1. 列出在控制台创建 VPC 时要填写的关键字段。
2. 如何验证一个实例是否可以访问互联网？

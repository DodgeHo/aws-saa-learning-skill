## 学习目标

- 了解 AWS 认证路径（从入门到专业）及每条路径的目标与适用人群。

## 重点速览

- AWS 认证分为 Foundational、Associate、Professional 与 Specialty，常见路径包括 SAA（Solutions Architect）、Developer、DevOps 等。

## 详细内容

1. 分级说明：Foundational（CLF-C01）适合入门；Associate 级别（SAA、DVA、SOA）侧重实践能力；Professional 与 Specialty 深入某一领域或角色。
2. 选择路径：根据职业目标、现有经验与学习成本选择合适的认证序列；可以先通过 Associate 级别建立基础再进阶。

## 自测问题

1. 列出 AWS 认证的四个等级并简短说明其定位。
2. 对于希望从事云架构职业的人，为什么建议先考 SAA？

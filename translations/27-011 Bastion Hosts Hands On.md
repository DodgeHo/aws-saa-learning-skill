## 学习目标

- 实操部署 Bastion Host 并验证通过它访问私有子网实例的流程。

## 重点速览

- Hands On 包括：在公有子网部署 EC2（禁用宽泛的入站规则）、配置安全组并测试通过 SSH 代理到私有实例。

## 详细内容

1. 步骤：创建公有子网 -> 启动 Bastion EC2 并分配 EIP -> 配置安全组仅允许管理 IP -> 使用 Bastion 作为跳板访问私有实例。
2. 验证：从管理机 SSH 到 Bastion，再从 Bastion SSH 到私有实例；记录连接日志以便审计。

2. 列出在部署过程中应记录的审计信息。

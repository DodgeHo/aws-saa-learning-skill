# EBS 实操

## 学习目标
- 创建并挂载 EBS 卷
- 理解 AZ 必须一致的限制
- 观察卷状态与实例挂载关系

## 重点速览
- 卷与实例必须在同一 AZ
- 创建后挂载到实例，状态变为 in-use
- 可用于故障切换（卸载再挂载）

## 详细内容
在 EC2 控制台创建 EBS 卷：
1. 进入 **Volumes**，选择 **Create volume**。
2. 类型选择 `gp2`/`gp3`，大小如 `2 GiB`。
3. **Availability Zone** 选择与实例一致的 AZ。
4. 创建后等待状态变为 `available`。

挂载到实例：
1. 选择卷，点击 **Attach volume**。
2. 选择目标实例，设备名使用默认 `/dev/sdf`。
3. 挂载后卷状态变为 `in-use`，实例存储中可见新卷。

用途提示：
- 可卸载后挂载到其他实例，适合故障切换
- 不使用时可删除卷以停止计费

## 学习目标

- 了解 AWS Backup 的功能、策略管理与跨服务备份方案。

## 重点速览

- AWS Backup 提供集中化备份策略，可对 EBS、RDS、DynamoDB、EFS、Storage Gateway 等服务进行备份与生命周期管理。

## 详细内容

1. 功能：集中备份计划、备份保留策略、跨区域复制与备份恢复；可通过标签进行策略应用。
2. 设计注意：制定保留策略以满足合规、评估恢复时间目标（RTO）与恢复点目标（RPO），并监控备份作业成功率与成本。

## 自测问题

1. 如何使用 AWS Backup 为 RDS 实例设置自动备份与跨区域复制？
2. 在备份策略中如何权衡保留期限与存储成本？

**学习目标**
- 了解 AWS App Runner 的定位与典型用例
- 掌握从源码或容器镜像快速部署 Web 应用的基本配置项

**重点速览**
- App Runner 是全托管服务，能从源码或容器镜像自动构建并部署 Web 应用与 API
- 提供自动扩缩、内置负载均衡、TLS、日志与可观察性；可配置 VPC 访问
- 适用于快速将 Web 服务推向生产的场景，运维复杂度低

**详细内容**
App Runner 旨在简化 Web 应用或 API 的部署：用户指定源代码仓库或容器镜像，选择 vCPU 与内存、端口与运行时配置，App Runner 会自动构建（若是源码）并创建托管运行环境、负载均衡与域名。服务支持水平自动扩缩、健康检查、日志与指标，并可配置访问私有 VPC 资源（如数据库）。

适用场景包括快速原型、小型微服务或希望将代码/镜像快速上线而不管理底层基础设施的团队。

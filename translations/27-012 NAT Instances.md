## 学习目标

- 理解 NAT 实例的作用、配置与与 NAT Gateway 的区别。

## 重点速览

- NAT 实例用于私有子网实例访问互联网（出站），它是用户管理的 EC2 实例，需要配置弹性 IP、路由及安全组。

## 详细内容

1. NAT 实例是自管解决方案：可自定义，成本较低但需负责可用性（需设置自动故障转移/AMI 等）。
2. 与 NAT Gateway 比较：NAT Gateway 是 AWS 托管、可扩展且更高可用但成本更高；NAT 实例灵活但需运维。

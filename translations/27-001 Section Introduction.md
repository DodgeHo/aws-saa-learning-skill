## 学习目标

- 了解本章涵盖的 VPC 与网络核心概念与 AWS 实践要点。
- 明确本章各节之间的关系以及实际考试/实操中需要掌握的主题。

## 重点速览

- 本章包含 VPC、子网、路由、NAT、Bastion、VPN、Direct Connect、Transit Gateway、IPv6 等关键网络主题。
- 侧重设计可用性、可扩展性与安全性，并强调与本地网络互联时的冲突避免与成本考量。

## 详细内容

1. 本章目标是把握 AWS 虚拟网络（VPC）构建与互联的整体流程和常见组件。
2. 每节包括概念说明、配置示例（Hands On）与设计建议，便于在考试与实际项目中应用。
3. 推荐学习顺序：VPC -> Subnet -> Route/IGW/NAT -> Bastion -> Peering/Endpoints -> VPN/Direct Connect -> Transit Gateway -> 辅助主题。

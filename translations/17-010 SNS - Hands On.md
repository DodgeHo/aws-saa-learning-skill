```markdown
**学习目标**

- 熟悉在控制台中创建 SNS 主题与订阅的基本步骤，并能验证订阅交付。
- 理解常见配置选项：主题类型（Standard / FIFO）、加密、访问策略与订阅验证流程。

**重点速览**

- 控制台流程：创建主题 → 创建订阅（选择端点类型：Email/SQS/Lambda/HTTP/SMS 等）→ 验证订阅（邮件/端点确认）→ 发布测试消息。
- FIFO 主题要求订阅为 SQS FIFO，主题名须以 `.fifo` 结尾；Standard 主题支持高吞吐与多终端订阅。

**详细内容**

- 操作要点：
  - 创建主题时设置加密（SSE/KMS）与访问策略（允许特定服务或账户发布）。
  - 订阅后大多数端点需要确认（Email、HTTP(S) 等），SQS/Lambda 等会自动确认并开始接收消息。
  - 发布消息可通过控制台测试，验证消息是否按期望投递到所有订阅端点；可结合订阅过滤策略只让子集接收消息。
  - 清理：测试完毕请删除订阅与主题以避免产生不必要通知或费用。

**自测问题**

1. 创建 SNS FIFO 主题时有哪些命名要求？只能订阅哪些类型的端点？
2. 如何在控制台验证 Email 订阅是否生效？
```

教师：AWS上的另一个机器学习服务叫做Amazon

Kendra｡

这是一个完全托管的文档搜索服务,

由机器学习提供支持,

它允许您从文档中提取答案｡

该文档可以是文本､ PDF､ HTML PowerPoint､

Microsoft Word､ 常见问题解答等｡

因此, 您有很多数据源,

这些文档可能就在其中,

您现在可以在屏幕上看到其中的一些, Amazon

Kendra将对它们进行索引, 它将在内部构建一个由机器学习支持的知识索引｡

从最终用户的角度来看, 它有何帮助？

我们有自然语言搜索功能, 就像你上谷歌一样｡

例如, 如果用户说, 嘿,

Amazon Kendra的IT支持台在哪里？

肯德拉可以回复, 在一楼.

这可能是因为Kendra从所有资源中了解到IT支持台在一楼,

这非常棒｡

而且, 你可以只做一个普通的搜索,

它会从用户的交互和反馈中学习, 以促进首选的搜索结果,

这被称为增量学习｡

最后, 您可以微调搜索结果, 例如,

基于重要数据､ 数据的重要性､

新鲜度或任何自定义过滤器, 好吗？

因此, 从考试的角度来看,

每当你看到一个文档搜索服务, 想想亚马逊肯德拉｡

就这样, 我们下节课再见｡

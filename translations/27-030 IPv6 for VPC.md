## 学习目标

- 了解在 VPC 中启用与使用 IPv6 的方法、优点与兼容性考虑。

## 重点速览

- IPv6 提供几乎无限地址空间，无需 NAT，但需考虑服务兼容性、路由与安全策略更新。

## 详细内容

1. AWS 支持为 VPC 分配 IPv6 CIDR（Amazon 提供或自带），子网可启用 IPv6 地址并配置路由表。 
2. IPv6 不需 NAT，简化端到端可达性；但某些 AWS 服务或第三方软件可能对 IPv6 支持不完整，需逐项验证。

## 自测问题

1. 启用 VPC IPv6 时常见的兼容性风险有哪些？
2. 描述 IPv6 在减少运维复杂性（如 NAT）方面的优势。

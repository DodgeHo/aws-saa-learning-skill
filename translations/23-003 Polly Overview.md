## 学习目标

- 了解 Amazon Polly 的用途：将文本合成为自然语音（TTS）并支持多语种与语音自定义。 
- 掌握 Polly 的主要 API 功能：实时合成、批量合成与语音格式选择。 

## 重点速览

- Polly 提供多种语音与语言、SSML 支持以及 Neural TTS 以提升自然度；常用于语音客服、无障碍与媒体朗读。 
- 支持将音频输出存储到 S3 或直接流式传输到客户端。 

## 详细内容

- 使用方式：
  - 通过 SDK/REST 调用 Polly，将文本或 SSML 发送以获取音频流或文件；可选择音色、语速与发音参数。 
  - 支持实时合成用于交互式应用，或批量合成用于内容生成与离线播报。 

- 集成与成本：
  - 可与 Amazon Lex、Lambda、S3 集成；计费基于合成的字符数与音频存储/传输。

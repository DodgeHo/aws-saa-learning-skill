# 创建 AWS 账号

## 学习目标
- 熟悉 AWS 账号创建的完整流程
- 理解免费套餐、信用卡验证与支持计划
- 了解激活与首次登录的注意事项

## 重点速览
- Root 账号用于登录与初始配置，密码务必妥善保管
- 信用卡验证是授权，不是收费
- 选择基础支持（Basic Support）即可

## 详细内容
创建 AWS 账号的第一步是填写用于登录的主邮箱地址，并为账号命名（例如 `stephane-demo-v3`）。完成邮箱验证后，需要设置并确认 root 用户密码。忘记密码可以通过邮箱找回，但最好的做法是妥善保存邮箱与密码。

接下来选择账号类型（个人账号），填写个人信息。随后需要填写信用卡信息，AWS 会进行 1 美元的授权验证，这是一笔待取消的授权，不是实际收费。只要保持在免费套餐范围内就不会收费；超过免费额度则会产生费用，课程里会讲如何监控成本。

然后填写手机号，使用短信或语音完成身份验证。支持计划选择基础支持即可。提交后账号会在几分钟内激活，并通过邮件通知。

激活完成后，进入 AWS 管理控制台，使用 root 邮箱与密码登录。账号创建成功，就可以开始学习了。

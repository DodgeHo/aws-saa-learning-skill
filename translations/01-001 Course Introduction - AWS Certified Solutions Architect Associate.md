# 课程介绍：AWS 解决方案架构师助理（SAA-C03）

## 学习目标
- 了解课程范围、时长与学习方式
- 认识讲师背景与课程覆盖的主题深度
- 初步理解 AWS 与传统数据中心的差异

## 重点速览
- 本课程围绕 SAA-C03 考试，覆盖通过考试所需知识
- 课程约 25 小时，会深入 30+ 项 AWS 服务
- 建议具备基本 IT 常识，但课程会从基础讲起
- 学习进度可放慢，反复观看完全正常

## 详细内容
欢迎加入 AWS 认证解决方案架构师助理课程。我是讲师 Stéphane Maarek，很高兴与你一起学习。本课程将针对 SAA-C03 考试，内容完整且系统，虽然时长较长（约 25 小时），但我会尽力让学习过程有趣且高效。

如果你具备一些 IT 基础会更轻松，但不用担心，我会一步步带你掌握通过认证所需的 AWS 知识。课程中会复用我在云实践者、开发人员与 SysOps 课程中的内容，因为这些认证之间有大量知识重叠。若你听到一些看似“别的考试”的关键词，请放心，你来对地方了。

本课程会深入讲解解决方案架构相关内容，并覆盖 30 多项 AWS 服务。即使你是初学者，也可以慢慢学，这不是比赛。需要时复习或二刷完全正常，很多学生都会这样做。

关于我：我刚通过 SAA-C03 考试，成绩 96%。我还持有多项 AWS 认证，曾担任 IT 顾问、解决方案架构师、开发与运维相关角色，并在云上构建过网站、应用和流媒体平台。我也教授过多门认证课程，涉及 CloudFormation、Lambda、EC2 等内容。

接下来我们会快速了解 AWS 是什么。AWS 是 Amazon Web Services，一家云服务提供商，提供按需、可扩展的服务器与服务。与传统数据中心相比，你不再需要采购和安装硬件，而是可以随时按需获取资源，并轻松扩展规模。AWS 已支持大量全球网站与业务（例如 Netflix），它让企业能够以更低成本、更高弹性运行服务。

AWS 的服务像一碗“意大利面”一样互相关联。我的目标是带你逐个拆解，最终把这碗“意大利面”吃透。很高兴与你一起学习，我们下节课见。

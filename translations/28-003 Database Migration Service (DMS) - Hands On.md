## 学习目标

- 通过实操掌握使用 DMS 创建复制实例、配置端点与迁移任务的基本步骤。

## 重点速览

- Hands On 包括：设置复制实例、配置源/目标端点、运行全量迁移或启用 CDC 并验证数据一致性。

## 详细内容

1. 步骤概览：准备源/目标（确保网络与权限）-> 创建 DMS 复制实例 -> 配置端点并测试连接 -> 创建迁移任务并选择迁移类型 -> 监控任务与验证数据。
2. 验证要点：检查表行数、关键数据字段、一致性校验；若为异构迁移，先使用 AWS SCT 转换 schema 并手动迁移不支持的对象。

1. 在 Hands On 中如何验证 CDC 是否正确捕获并复制变更？
2. 列出迁移任务失败时常见的三个排查方向。

## 学习目标

- 理解 Amazon Personalize 的定位：托管推荐引擎服务，支持个性化推荐与排行优化。 
- 掌握数据准备、方案训练、评估与部署推荐策略的基本流程。 

## 重点速览

- Personalize 提供端到端推荐流水线（数据导入、训练、活动流实时推荐），适用于电商、内容推荐等场景。 
- 支持多种推荐算法与在线/离线评估，能够通过 A/B 测试优化效果。 

## 详细内容

- 流程与要点：
  - 数据要求包括用户、项目、交互数据；通过导入数据集并训练解决方案（Solution）来生成模型。 
  - 部署后可通过 GetRecommendations API 实现实时推荐，也可批量生成推荐列表。 

- 监控与迭代：
  - 使用离线指标（精确率、召回率）与在线实验（A/B）评估并迭代推荐策略；注意冷启动问题与数据偏差。

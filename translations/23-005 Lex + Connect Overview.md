## 学习目标

- 了解 Amazon Lex（对话式 AI）与 Amazon Connect 的结合场景与基本架构。 
- 掌握构建语音/文本机器人、集成语音渠道与将 Lex 用于 IVR 或客服自动化的要点。 

## 重点速览

- Lex 提供 NLU 与对话管理能力，Connect 是托管联络中心服务；结合可实现自动应答、语音机器人与数据收集。 
- 常见集成：Lex 识别意图并触发 Lambda，Connect 管理呼叫路由与录音。 

## 详细内容

- 构建对话机器人：
  - 使用 Lex 定义意图、槽位与对话流程；可使用 Lambda 函数执行后端逻辑与外部 API 调用。 
  - 在 Connect 中创建联络流程并将 Lex 机器人嵌入以提供自动化语音交互。 

- 监控与优化：
  - 使用 CloudWatch、Connect 报表与实时指标监控交互质量；通过对话日志与意图命中率迭代优化。

# 路由策略：基于 IP

## 学习目标
- 理解基于 IP 的路由逻辑
- 掌握 CIDR 匹配机制

## 重点速览
- 根据客户端 IP 所属 CIDR 路由
- 适合已知来源流量优化

## 详细内容
基于 IP 路由使用 CIDR 列表：
- 不同 CIDR 对应不同记录/端点
- 适合特定 ISP 或固定来源流量

场景：
- 通过已知来源优化性能或成本

## 自测问题
- 为什么需要 CIDR 而不是单个 IP？
- 基于 IP 路由适合哪类业务场景？

# Route 53 概览

## 学习目标
- 了解 Route 53 的定位与特点
- 认识常见记录类型
- 区分公共/私有托管区域

## 重点速览
- Route 53 是权威 DNS，支持域名注册
- 支持健康检查与多种路由策略
- 公共/私有托管区域用途不同

## 详细内容
Route 53 特性：
- 高可用、可扩展、完全托管、权威 DNS
- AWS 唯一提供 100% 可用性 SLA 的服务
- 端口 53 为 DNS 标准端口

核心记录字段：
- 记录名、类型、值、路由策略、TTL

常见记录类型：
- **A**：域名 -> IPv4
- **AAAA**：域名 -> IPv6
- **CNAME**：域名 -> 域名（不能用于 Zone Apex）
- **NS**：托管区域名称服务器

托管区域：
- **Public Hosted Zone**：对公网解析
- **Private Hosted Zone**：仅在 VPC 内解析

费用提示：
- 托管区域按月计费
- 域名注册按年计费

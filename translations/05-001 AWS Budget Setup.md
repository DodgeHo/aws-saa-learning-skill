# AWS 预算与账单监控

## 学习目标
- 打开 IAM 对计费数据的访问权限
- 使用账单与免费层页面定位费用来源
- 创建预算与费用预警

## 重点速览
- 默认情况下 IAM 用户无法访问账单信息
- 免费层页面可查看当前与预测使用量
- 建议设置 0 美元或低额预算以防误操作

## 详细内容
为了避免产生意外费用，建议先配置预算与告警。进入 **Billing and Cost Management**（计费与成本管理）时，如果你使用的是 IAM 用户，会看到访问被拒绝。需要用 **root 用户**进入账户设置，启用“**IAM 用户和角色访问计费信息**”。启用后，IAM 管理员即可查看账单。

在账单页面可以看到本月成本、预测成本以及按服务拆分的费用明细。若出现费用，进入对应月份的“按服务计费”可以定位是哪项服务产生了成本。

“免费层（Free Tier）”页面会显示当前与预测使用量。若预测使用量进入红色区域，代表即将超出免费额度，需要及时停用资源。

预算设置建议：
- **零支出预算**：达到 0.01 美元即告警。
- **月度成本预算**：例如每月 10 美元，上限达到 85%/100% 及预测超支时触发通知。

这些工具可以帮助你在课程学习过程中及时发现成本问题。

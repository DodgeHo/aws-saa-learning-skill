## 学习目标

- 理解 Amazon Translate 的用途：自动化文本翻译服务，支持多语言与批量翻译场景。 
- 掌握 API 调用模型、批量翻译流程与简单的术语表/自定义翻译选项。 

## 重点速览

- Translate 提供实时与批量文本翻译，适合本地化、跨语言客服与多语言内容处理。 
- 支持自定义术语表以保持行业术语一致性，并能与其他服务协同（S3、Lambda）。 

## 详细内容

- 功能与用法：
  - 使用 SDK 调用 Translate 进行单句或批量翻译；可指定源语言或让服务自动检测。 
  - 支持自定义术语表和并行批处理作业来处理大规模文本。 

- 质量与治理：
  - 自动翻译适合一般沟通与非敏感内容；对法律/医药等高风险文本应结合人工校验或专业翻译。

# EFS vs EBS

## 学习目标
- 对比 EFS 与 EBS 的核心差异
- 能根据场景选择合适的存储

## 重点速览
- EBS：块存储，单实例挂载，AZ 绑定
- EFS：文件存储，多实例共享，跨 AZ
- EFS 成本更高，但适合共享场景

## 详细内容
**EBS（块存储）**
- 一次挂载到一个实例
- 绑定单个 AZ
- 低延迟随机 I/O
- 成本相对较低

**EFS（文件存储）**
- 多实例同时挂载
- 跨多个 AZ，高可用
- 适合共享数据、内容管理
- 成本更高

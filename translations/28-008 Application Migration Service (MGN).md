## 学习目标

- 理解 AWS Application Migration Service（MGN，又名 AWS Server Migration 的后续）用于服务器迁移的流程与优点。

## 重点速览

- MGN 支持将本地物理/虚拟服务器无缝迁移到 AWS，自动复制并在目标上启动测试实例，减少停机时间。

## 详细内容

1. 迁移流程：安装复制代理 -> 配置复制设置 -> 进行连续数据复制 -> 测试启动转换后的实例 -> 切换至 AWS 上运行。
2. 优点：自动化复制、最小化停机、支持多平台；注意网络带宽与目标实例大小的评估。

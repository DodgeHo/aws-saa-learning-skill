## 学习目标

- 了解在 AWS 环境中阻断或限制来自特定 IP 的访问的常见方法与最佳实践。

## 重点速览

- 常用方法包括 Security Groups、NACL、WAF 规则及 Network Firewall 策略，选择取决于作用范围（实例级、子网级、应用层或集中防护）。

## 详细内容

1. Security Group：适用于实例层的允许/拒绝规则（有状态），通常更适合白名单管理；不适合大规模 IP 黑名单。
2. NACL：子网层无状态规则，适用于阻断简单的 IP 段或临时封禁，但复杂规则管理不便。
3. AWS WAF：面向应用层（HTTP/HTTPS），支持 IP 集合、速率限制与复杂规则，可用于阻断恶意请求源。
4. AWS Network Firewall：用于网络层集中化策略，适合企业级复杂规则与日志审计需求。

# EBS 快照实操

## 学习目标
- 基于卷创建快照
- 用快照创建新卷
- 理解跨 AZ/Region 的迁移路径

## 重点速览
- 卷 -> 快照 -> 新卷
- 快照可复制到其他 Region
- 新卷创建时需指定 AZ

## 详细内容
创建快照：
1. 进入 **Volumes**，选择目标卷。
2. **Actions** -> **Create snapshot**。
3. 填写描述后创建，等待状态从 `pending` 变为 `completed`。

用快照创建新卷：
1. 进入 **Snapshots**，选择快照。
2. **Actions** -> **Create volume**。
3. 选择 AZ，创建后可挂载到实例。

跨 Region：
1. 选择快照，**Actions** -> **Copy snapshot**。
2. 选择目标 Region 后创建。

# EBS 快照

## 学习目标
- 理解快照的用途与增量特性
- 知道快照跨 AZ/Region 的迁移方式
- 了解归档与回收站功能

## 重点速览
- 快照是 EBS 卷的时间点备份
- 增量存储：只保存变更块
- 可复制到其他 AZ/Region 或共享给其他账号

## 详细内容
EBS 快照用于备份卷数据，支持跨可用区和跨区域迁移：
- **增量快照**：首次为全量，后续仅保存变化块
- **跨区域复制**：使用 Copy Snapshot 将备份迁移到其他 Region
- **共享与模板**：可共享给其他账号，也可用作 AMI 的底层卷

新特性：
- **Snapshot Archive**：归档层成本更低，但恢复时间更长（约 24-72 小时）
- **Snapshot Recycle Bin**：回收站防止误删，可设置保留期

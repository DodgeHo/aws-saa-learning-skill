# 关于控制台 UI 变化

## 学习目标
- 理解课程截图与当前界面可能不同的原因
- 了解如何在新旧 UI 之间切换
- 建立“界面变化不等于课程过时”的心态

## 重点速览
- AWS 正在逐步更新 UI，不同服务可能处于不同阶段
- 如果迷路，可以切回旧 UI
- 讲师会持续更新重大变更内容

## 详细内容
AWS 正在进行大规模的控制台界面升级，以适配新的设计规范。这一更新是分阶段进行的，不是所有服务同时更新，因此你可能会看到新旧 UI 混合存在。

我会尽力保持课程视频的更新，但在 AWS 完成全部迁移前，不可能逐一同步所有细小变化。如果界面差异不影响操作路径，我可能不会立即重录；若变化足以影响学习，我会更新对应视频。

如果你在新 UI 中找不到操作入口，很多服务允许切换回旧 UI，这通常会与你在课程视频中看到的界面更一致。

请放心，课程并不过时。我会持续监控变化并更新关键内容。感谢理解，祝你学习愉快。

## 自测问题
- 为什么 AWS 控制台会出现新旧 UI 混合？
- 如果找不到入口，你可以采取的第一步是什么？

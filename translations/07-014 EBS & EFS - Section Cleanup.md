# EBS 与 EFS 清理

## 学习目标
- 完成实验资源清理
- 避免不必要的费用

## 重点速览
- 删除 EFS 文件系统
- 终止实例并删除卷/快照
- 清理非默认安全组

## 详细内容
清理建议按顺序进行：
1. **删除 EFS 文件系统**：进入 EFS，删除时需输入文件系统 ID 确认。
2. **终止 EC2 实例**：确保所有实验实例已停止并终止。
3. **删除 EBS 卷**：删除所有不再使用的卷。
4. **删除快照**：清理实验快照，避免存储计费。
5. **清理安全组**：仅删除非默认安全组；若被实例引用则需先终止实例。
